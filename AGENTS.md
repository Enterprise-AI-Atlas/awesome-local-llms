# AGENTS.md — Awesome Local LLMs

## Purpose

This repository is a curated registry of tools, models, and frameworks for running large language models locally or on-premises.

## Scope

Cover resources that enable local/private LLM inference and serving:

- Local LLM inference engines and runtimes.
- Quantized model formats and conversion tools.
- On-premise LLM deployment frameworks.
- Local chat UIs and clients.
- Hardware guidance for local LLM workloads.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the runtime, model format, or deployment category they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
